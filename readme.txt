To run the D&D Tracker Application:
1) Go to https://github.com/ajnahat/DnDTrackerAPI and click the green 'Code' button, and download the zip. Extract to the downloads folder.
2) Repeat step 1 for https://github.com/ajnahat/DnDTracker. There should now be a DNDTrackerAPI folder and a DNDTracker folder in the downloads folder.
3a) If Docker is not installed, download it here https://www.docker.com/products/docker-desktop and install it.
3b) Run this command in a command prompt without quotes to clone the docker container "docker pull ajnahat/dndtracker".
4) Run the two .bat files in the "Run" folder in order.
